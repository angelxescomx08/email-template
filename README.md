# Template para diseños de correos

El template es básico y tiene las metatags necesarias para compatibilidad. El template usa estilos con la etiqueta style, esto no es recomendable por lo que se debería usar estilos en linea, usar alguna herramienta para esto.

No utilizar flexbox, ni grid, los diseños deben ser desarrollados con tablas. Las imágenes deben estar subidas a un servidor al igual que las fuentes (poner un fallback por si el servidor de correo no proporciona esta funcionalidad).
